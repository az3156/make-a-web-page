Lesson 1

HTML and the Internet

HTML(Hyper Text Markup Language)is the language computers use in Web browsers to display content over the Internet.

Elements

There are two kinds of elements:inlineandblock.

Inline elements will display text on the same line unless wrapping to the next or using a break tag.

When multiple spaces or line breaks are visible in the HTML editor, they won't be visble on the web page using inline elements. This white space is treated as a single space.Blockelements will contain text inside an invisible box.

Tags

Tags inform the web browser how to display the contents inside of the tags.

The content can consist of text, image, or a URL.

Most elements require a start and an end tag. These are formatted with a letter or letters inside of the left < and right > arrow symbols. Some tags require an attribute and associated parameter in order for the tag to fully function.

Inline examples: b=bold, em=emphasis/italics, a=anchor, span, strong
Note: br=line break is an inline void element, which does not require an end tag.
Block tag examples: p=paragraph, div, form

Lesson 2

Structure

Web pages are built with arectangularstructure that consists ofboxes.Boxes contain varying elements such as text or images and can vary in size and formatting.

HTML provides the structure for a web page, while CSS provides the style for how it looks.

Visualization

When viewing a graphical design for a web page, it helps to divide it into boxes that separate the content and structure for the page. Start the process by dividing it into large boxes and work down to small boxes.

Visualize the structure like a tree that begins with a trunk or large section down to smaller branches. The larger branches contain main elements that group related content together in the larger boxes.

Lesson 3

Coding

The style coding should avoid repetition since it will reduce the chance of making errors and make for a cleaner document.

It's important to lay the code out in a nested or tree like manner. It greatly aids the programmer to structure the page properly and identify mistakes more easily.

Adding Style to HTML

CSS(Cascading Style Sheet)is an effective way to add style to an HTML file. It controls how the objects are displayed on the screen using an Internet browser. The document type should be defined in the HTML file to specify the overall rules for the style elements. Defining the HTML document type as HTML5 applies explicit rules to the file that allow the style to manipulate the display. It can be defined at the beginning of the HTML file as follows:
!DOCTYPE html
<html>
(HTML content)
</html>

Methods for adding Style

Style can be added to the HTML file inline, in its own section inside the HTML file or in a separate CSS file that can be linked to the HTML document. It's preferable to use a separate CSS file for adding style to HTML to keep the coding simpler and especially for larger, more complex files. Style elements can apply to larger sections and down to small selections. Linking CSS is done in the head tag section using the following example:
<link rel="stylesheet" type="text/css" href="style.css">
Here's an example of adding style inline:
<div style="color: red">

CSS files contain references to the HTML elements in which they apply style. These references are followed by properties and values inside of curly brackets that define the style for that particular element. For example, if you want to change the font or font color of the h1 element, it would look like this:
h1 {
color: red;
}

Boxes

The boxes that contain an element can be manipulated in the CSS style file. The box size and layout on the screen can be changed by adding properties in the CSS file.

The div element is a block tag. However, it can be changed to appear inline by adding the flex value to its display property. This allows the div elements to appear on screen side-by-side as follows:
<div class="app" style="display: flex;>

The values for box sizing can be done by defining the pixel width or heighth. However to properly adjust when the browser screen adjusts, the values can be defined by percentages in addition to maximum pixels. Example:
width: 50%;
height: 50%;
max-width: 705px;
